# Leaderboard

> The leaderboard website displays scores submitted by different players. It also allows you to submit your score. All data is preserved thanks to the Leaderboard API service.

## Built With

- HTML5
- CSS4
- JavaScript ES6
- Webpack

## Live Demo (if available)

[Live Demo Link](https://livedemo.com)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Visual Studio code or any Code Editor.

### Setup

#### Download the file to local directory

- if you have already insall github in your local pc run:

```sh
git clone git@github.com:darikmohammed/Leaderboard.git
```

- Or download the zip file from the github directory.

#### Linters tests

- Lighthouse
- Webhint
- Stylelint
- node_modules checker
